Delectable Options

You need to enter your username and password for a user on your WordPress site. The XML-RPC Endpoint will be the location of your WordPress blog, such as http://www.mydomain.com/xmlrpc.php

(Do NOT forget to include the xmlrpc.php at the end of the path)

Advanced Options

If you leave the category name blank, the new WordPress 3.1 post format 'link' will be used.

If you leave the Post Type blank, the default of 'post' will be used. 'page' is a valid option if you wish all of your links to be pages. Custom Post Types are currently NOT supported by the WordPress API. So, at the moment you are limited to 'post' or 'page'
